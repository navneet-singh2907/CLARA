# Sample Loan Documents

Use these files for demo uploads in the CLARA web UI.

## Northstar Custom Cabinets

File:

```text
Northstar_Custom_Cabinets_Loan_Profile.pdf
```

This is a synthetic SBA-style borrower packet created for portfolio demonstration. It includes:

- Borrower and company details
- Loan amount, SBA guarantee, term, jobs supported, and NAICS
- Credit score, years in business, prior default status, and missing documents
- Financial snapshot and documentation checklist
- Machine-readable intake fields for reliable CLARA document parsing

Expected review theme:

```text
Generally viable borrower with stable operating history, but missing beneficial ownership certification should trigger documentation review.
```

This is not a real borrower file and should not be used for actual credit decisions.
